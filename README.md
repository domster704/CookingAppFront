# GameRecipes

Веб-приложение с использованием **sber assistant** и с возможностью поиска и добавления рецептов из различных игр и произведений. 

Для работы используются _Studio Canvas App_ и _Studio Code App_ от Сбера.

## Стек технологий
- ReactJS, JS
- Webpack
- Babel
- HTML, CSS
- Sber Assistant
